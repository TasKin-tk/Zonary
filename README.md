## English

### Description

Zonary is a minimalist binary-based esoteric programming language created by TasKin.  
It features no negative numbers, registers named by 8-bit binary numbers, and a fixed-length instruction set.  
It is designed to explore the boundaries of minimalism in language design.  

### Features

- **No Negative Numbers**: All values are in the range 0-255, embracing the beauty of unsigned integers.  
- **Binary Registers**: Registers are named using 8-bit binary numbers (e.g., `00001010`), pushing the limits of memorization.  
- **Fixed-Length Instructions**: Every instruction has a fixed bit length, making parsing straightforward (or tedious).  
- **Pure Binary Code**: Code is composed entirely of 0s and 1s; spaces and newlines are ignored.  
- **Esoteric Fun**: A language designed to be challenging and amusing to write, not for practical use.  

### Files in the Repository

This repository contains the Zonary interpreter, documentation, and a test Zonary file.  
- `Zonary-version.py` - The Zonary interpreter (Python 3).  
- `Zonary-Documentation.txt` - The complete language specification.  
- `test.zonary` - A Zonary file for testing.

```
git clone https://github.com/TasKin-tk/Zonary.git
```

## 中文

### 简介

Zonary 是一个基于二进制的极简整活编程语言，由 TasKin 制作。  
它没有负数，寄存器使用 8 位二进制数字命名，并且所有指令长度固定。  
它的设计目的是探索语言设计中极简主义的边界。  

### 特点

没有负数：所有值都在 0~255 之间，拥抱无符号整数的纯粹。  
二进制寄存器：寄存器使用 8 位二进制数命名（例如 00001010），挑战你的记忆极限。  
定长指令：每条指令长度固定，解析简单（且枯燥）。  
纯二进制代码：代码全部由 0 和 1 组成，空格和换行会被忽略。  
整活乐趣：一门旨在挑战书写难度和带来乐趣的语言，不适合实际使用。
